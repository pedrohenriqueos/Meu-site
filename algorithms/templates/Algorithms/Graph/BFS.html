Busca em Largura

Em Grafos a busca em largura busca visitar todos os nós que estão ligados nele, antes de passar para uma próxima camada de busca, para isso é necessário armazenar a ordem em que eles serão visitados, para manter a ordem dos elementos utiliza-se uma fila para manter a ordem dos nós que devem ser visitados. Gerando o código: 

                    
vector< int > G[10000];
int cor[10000], dist[10000];
void bfs(int v){
	queue q;
	q.push(v);
	cor[v]=1;
	while(!q.empty()){
		int u=q.front();
		q.pop();
		for(int w:G[u])
            if(!cor[w]){
                q.push(w);
                cor[w]=1;
                dist[w]=dist[u]+1;
            }
	}
}
