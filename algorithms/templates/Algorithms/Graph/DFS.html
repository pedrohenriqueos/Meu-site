Busca em Profundidade

Em Grafos a busca em profundidade visita um nó e busca o próximo e visita o próximo encontrado, assim que ele é encontrado. Quando ele não possui mais nenhum nó para ser visitado ele retorna para o nó que era o pai deste visitado e continua buscando novos nós que ainda não foram visitados. Gerando o código: 

                    
vector< int > G[10000];
int cor[10000], dist[10000];
void dfs(int v){
    cor[v] = 1;
    for(int u:G[v])
        if(!cor[u]){
            dist[u] = dist[v] + 1;
            dfs(u);
        }
}
