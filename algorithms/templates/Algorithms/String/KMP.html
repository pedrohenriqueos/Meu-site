KMP

KMP é um algoritmo de matching em string, com o objetivo de encontrar a posição do matching ou quantas vezes aquela substring existe dentro de outra string. Aparenta ser um problema simples, pois sua solução trivial é O(n²), porém a solução de simplesmente verificar todas as possibilidades de matching não é uma forma inteligente, e este algoritmo possui uma solução para o problema na complexidade de O(n).

Para iniciar o problema vamos nomear as strings, nós temos o nosso texto e o padrão buscado.
