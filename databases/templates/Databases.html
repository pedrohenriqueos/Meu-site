Banco de Dados

Existem dois tipos de banco de dados, o relacionado e o não relacionado. O relacionado são bancos organizados em tabelas, como exemplo temos: Mysql, Postgresql e Aurora. Este tipo de banco de dados possui uma forma de manipulação de dados (Data Manipulation Language, DML) utilizando os comandos:

  • SELECT: Usado para selecionar um subgrupo de dados que estão presentes dentro do banco de dados;
  • UPDATE: Utilizado para atualizar um ou mais dados do banco;
  • INSERT: Adiciona novos dados dentro do banco de dados;
  • DELETE: Apaga um ou mais elementos do banco de dados.

E a forma de construção e definição (Data Definition Language, DDL) de um banco de dados é feito pelos comandos:

  • CREATE: cria uma nova tabela, um novo banco ou um novo usuário;
  • ALTER: alterar a coluna de uma tabela;
  • DROP: deleta uma tabela;
  • TRUNCATE: deleta todos os dados de uma tabela.
